## Prompts for Ad Copy

### Prompt 1: Generate Hook Variations

```
I'm creating Facebook video ads for [PRODUCT] which [WHAT IT DOES].

Target audience: [DESCRIBE AUDIENCE]
Main pain point: [PAIN POINT]
Key benefit: [BENEFIT]

Generate 20 different video hook options (first 3 seconds) using these styles:
1. Question hooks (5)
2. Problem call-out hooks (5)
3. Curiosity/intrigue hooks (5)
4. Social proof hooks (5)

Each hook should be under 10 words and stop the scroll immediately.
```

### Prompt 2: Full UGC Script

```
Write a 20-second UGC-style video ad script for [PRODUCT].

Product: [NAME AND DESCRIPTION]
Price point: [PRICE]
Target customer: [WHO]
Main problem it solves: [PROBLEM]
Key differentiator: [WHAT MAKES IT DIFFERENT]

Use this structure:
- Hook (0-3 sec): Attention grabber
- Problem (3-8 sec): Relatable pain point
- Solution (8-15 sec): Show the product working
- CTA (15-20 sec): Clear call to action

Write it in casual, conversational language like a real person talking to their phone. Include [VIDEO DIRECTION] notes.
```

### Prompt 3: Ad Copy Variations (AIDA)

```
Create 5 Facebook ad copy variations for [PRODUCT].

Product: [DESCRIPTION]
Offer: [CURRENT OFFER]
Target: [AUDIENCE]
Tone: [CASUAL/PROFESSIONAL/URGENT]

For each variation, use the AIDA framework:
- Attention: Hook headline (under 40 characters)
- Interest: Relatable problem/situation
- Desire: Key benefit and transformation
- Action: Clear CTA

Include:
- Primary text (under 125 words)
- Headline (under 40 characters)
- Description (under 30 words)
```

### Prompt 4: Carousel Story Arc

```
Create a 5-card carousel ad story for [PRODUCT].

Product: [DESCRIPTION]
Target: [AUDIENCE]
Goal: [CONVERSIONS/AWARENESS/TRAFFIC]

Structure:
Card 1: Hook - Stop the scroll with a question or bold statement
Card 2: Problem - Expand on the pain point
Card 3: Solution - Introduce the product
Card 4: Proof - Social proof or key benefit
Card 5: CTA - Offer and call to action

For each card provide:
- Headline text (under 25 characters)
- Body text (under 50 characters)
- Image/visual description
```

### Prompt 5: Objection-Handling Ads

```
My product [PRODUCT] has these common objections from customers:
1. [OBJECTION 1]
2. [OBJECTION 2]
3. [OBJECTION 3]

For each objection, create:
- A hook that acknowledges the objection
- A response that overcomes it
- A CTA that reduces risk

Format as short video script (15 seconds each).
```

## Prompts for Creative Strategy

### Prompt 6: Angle Brainstorming

```
I sell [PRODUCT] to [TARGET AUDIENCE].

Generate 15 different creative angles I could test, including:
- Problem-focused angles (5)
- Benefit-focused angles (5)
- Lifestyle/identity angles (5)

For each angle, provide:
- The angle in one sentence
- A sample hook
- Why it might work
```

### Prompt 7: Competitor Ad Analysis

```
Analyze this competitor ad copy:

[PASTE COMPETITOR AD COPY]

Identify:
1. The hook they're using
2. The main pain point addressed
3. The value proposition
4. The CTA strategy
5. What's working well
6. What could be improved

Then suggest how I could create a better version for my product [YOUR PRODUCT].
```

### Prompt 8: Seasonal Campaign Ideas

```
I sell [PRODUCT] in the [INDUSTRY] space.

Create a content calendar for [MONTH/QUARTER] including:
- Relevant holidays or events
- Seasonal angles to test
- Promotional campaign ideas
- Creative themes that fit the season

For each idea, provide:
- Campaign name
- Key message
- Suggested offer
- Creative format (video/static/carousel)
```

## Prompts for Optimization

### Prompt 9: Ad Performance Analysis

```
Here are my ad metrics:
- CTR: [X]%
- Hook rate: [X]%
- CPC: $[X]
- Conversion rate: [X]%
- ROAS: [X]

Industry benchmarks are:
- CTR: 0.9-1.6%
- Hook rate: 25%+
- CPC: $0.70-1.50
- ROAS: 2.5-4.0

Analyze my performance and suggest:
1. What's working
2. What needs improvement
3. Specific actions to take
4. What to test next
```

### Prompt 10: A/B Test Ideas

```
My current winning ad has:
- Hook: [HOOK]
- Format: [VIDEO/STATIC/CAROUSEL]
- Angle: [ANGLE]
- CTA: [CTA]

Generate 5 meaningful A/B test ideas that change ONE significant element:
1. Different hook approach
2. Different format
3. Different angle/message
4. Different CTA
5. Different visual style

For each, explain the hypothesis and expected impact.
```

## AI Workflow for Volume Testing

### Step 1: Generate Concepts
Use AI to brainstorm 20+ angles/hooks in minutes.

### Step 2: Create Variations
For each winning concept, generate 5 variations.

### Step 3: Produce at Scale
Use AI design tools to create assets quickly.

### Step 4: Launch and Learn
Test all variations, let data decide winners.

### Step 5: Iterate Winners
Use AI to create variations of what's working.

## Best Practices

**Do:**
- Generate lots of options, test aggressively
- Use AI for first drafts, human for polish
- Feed AI your brand voice examples
- Test AI-generated vs human-created
- Use AI to speed up iteration cycles

**Don't:**
- Use AI output without review
- Rely on one AI tool only
- Skip human creative direction
- Forget brand guidelines
- Ignore platform-specific needs
