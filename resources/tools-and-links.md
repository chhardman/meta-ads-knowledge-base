## Competitor Research Tools

### Free
| Tool | Best For |
|------|----------|
| **Meta Ads Library** | See all active competitor ads |
| **Manual Feed Monitoring** | Save ads that appear in your feed |

### Premium Spy Tools
| Tool | Price | Best For | Link |
|------|-------|----------|------|
| **AdSpy** | $$$ | Largest historical database (177M+ ads) | [adspy.com](https://adspy.com) |
| **Minea** | $49/mo | Ecommerce/trending products, TikTok + Pinterest too | [minea.com](https://minea.com) |
| **MagicBrief** | $249/mo | DTC brands, AI-generated competitor summaries | [magicbrief.com](https://magicbrief.com) |
| **AdFlex** | $99/mo | Landing page and funnel analysis | [adflex.io](https://adflex.io) |
| **Dropispy** | $29/mo | Dropshipping focused | [dropispy.com](https://dropispy.com) |
| **BigSpy** | $9/mo+ | Budget-friendly option | [bigspy.com](https://bigspy.com) |

### Choosing the Right Tool
- **Just starting out:** Meta Ads Library (free)
- **Ecommerce/dropshipping:** Minea or Dropispy
- **DTC brands:** MagicBrief
- **Deep research:** AdSpy

## Tool Evaluation Checklist

Before adding a new tool, ask:

- [ ] Does it solve a real problem we have?
- [ ] Can we achieve this with existing tools?
- [ ] What's the ROI potential?
- [ ] Does it integrate with our current stack?
- [ ] Is there a free trial?
- [ ] What's the learning curve?
- [ ] Who on the team will own it?
