# Meta Ads Knowledge Base

A comprehensive marketing knowledge database for Meta (Facebook/Instagram) advertising.

## Quick Start

**To view locally:** Just open `index.html` in your browser.

---

## Deploy & Share (Free)

### Option 1: Netlify Drop (Easiest - 30 seconds)

1. Go to [netlify.com/drop](https://app.netlify.com/drop)
2. Drag the entire `Latest Meta Trends` folder onto the page
3. Done! You'll get a shareable URL like `random-name-123.netlify.app`
4. (Optional) Click "Set up your site" to customize the URL

### Option 2: GitHub Pages (Best for updates)

1. Create a GitHub account if needed
2. Create a new repository
3. Upload all files from this folder
4. Go to Settings > Pages
5. Set Source to "main" branch
6. Your site will be at `yourusername.github.io/repo-name`

### Option 3: Vercel

1. Go to [vercel.com](https://vercel.com)
2. Sign up with GitHub
3. Import your repository
4. Deploy automatically
