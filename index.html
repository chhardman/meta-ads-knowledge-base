UGC Video Scripts

Ready-to-use scripts. Hand these to creators.

Script Structure: H-P-S-C

All scripts follow: Hook ‚Üí Problem ‚Üí Solution ‚Üí CTA

  • Hook (0-3 sec): Stop the scroll
  • Problem (3-8 sec): Relatable pain point
  • Solution (8-15 sec): Product demonstration
  • CTA (15-20 sec): Clear next step

Script Library

Script 1: The Skeptic Turned Believer

Best for: Products with skeptical audiences, premium pricing

HOOK: "I'm not gonna lie, I thought this was just another overhyped [product]..."

PROBLEM: "I've tried literally everything for [problem]. Spent hundreds on stuff that didn't work."

SOLUTION: "But after using [Product] for [timeframe], I'm actually shocked. Look at this [specific result]."

CTA: "Link's in bio. They have [offer] right now."

Script 2: The Problem Call-Out

Best for: Products solving clear pain points

HOOK: "If you're still dealing with [specific problem], stop scrolling."

PROBLEM: "I know how frustrating it is when [describe frustration]. You've probably tried [common solution] and it didn't work."

SOLUTION: "This is [Product]. It actually [key benefit] because [simple reason]."

CTA: "They're doing [X]% off this week. Link in bio."

Script 3: The Comment Response

Best for: Addressing objections, building trust

HOOK: [Show comment: "Those are so overpriced"] "Okay I get this comment a lot, let me explain..."

RESPONSE: "So yes, [Product] is [price], but here's what you're getting... [quality/feature 1], [feature 2], and [feature 3]."

COMPARISON: "Compare that to [cheaper alternative] which is [problem with cheap option]."

CTA: "If you want to see for yourself, link's in bio."

Script 4: The Before/After

Best for: Transformation products

HOOK: [Split screen] "[Timeframe] with [Product]. Same lighting, no filter."

BODY: "Here's what I was dealing with before... And here's after [timeframe] of using [Product]..."

PROOF: "I was honestly surprised because I've tried [competitors] and nothing worked like this."

CTA: "Link below - they have a [guarantee] so you can try risk-free."

Common Problems & Fixes

Problem: Ads Not Delivering

Budget Too Low: Set budget at 10x your target CPA. If target is $50, daily budget should be $500+.

Audience Too Narrow: Facebook needs 1,000+ users. Expand geo, remove interest layers, test broad targeting.

Bid Too Low: Switch to Lowest Cost (automatic), remove bid caps temporarily.

Problem: Low CTR (<0.5%)
  • Hook not working ‚Üí Test 5-10 different hooks
  • Wrong audience ‚Üí Review demographics, test broader
  • Looks like an ad ‚Üí Use UGC-style, remove heavy branding
  • Weak value prop ‚Üí Clarify benefit, add social proof
Problem: Low Conversions
  • Landing page slow ‚Üí Speed should be <3 seconds
  • Message mismatch ‚Üí Ad promise must match page headline
  • Wrong optimization ‚Üí Switch to Conversions, not Traffic
  • Tracking broken ‚Üí Check Pixel + Conversions API

Campaign Launch Checklist

Before You Launch
  • ‚òê Clear campaign objective defined
  • ‚òê Budget set (10x target CPA minimum)
  • ‚òê 8-15 genuinely different ad concepts
  • ‚òê Multiple formats (video + static + carousel)
  • ‚òê Hook optimized for first 3 seconds
  • ‚òê Headline under 40 characters
  • ‚òê Pixel + Conversions API verified
  • ‚òê Landing page loads <3 seconds
  • ‚òê Message matches between ad and page
  • ‚òê Mobile tested

Weekly Optimization Checklist

Monday: Performance Review
  • ‚òê Review metrics vs. last week
  • ‚òê Check CTR trends (target: 0.9-1.6%)
  • ‚òê Review CPA/ROAS vs. target
  • ‚òê Check frequency (<3-4 ideal)
Tuesday: Creative Health
  • ‚òê Identify fatigued ads (declining CTR)
  • ‚òê Pause underperformers ($50-100 no results)
  • ‚òê Scale winners (20% max increase)
  • ‚òê Check competitor Ad Library

Scaling Checklist

Before Scaling
  • ‚òê Consistent performance 5+ days
  • ‚òê CPA at or below target
  • ‚òê Learning phase exited
  • ‚òê Frequency still healthy (<3)
  • ‚òê Increase budget 20% max per change
  • ‚òê Wait 2-3 days between increases
