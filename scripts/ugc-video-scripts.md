# UGC Video Scripts Library

> Ready-to-use scripts for your next ad. Adapt to your product/brand.

---

## Script Framework: H-P-S-C

All scripts follow: **Hook → Problem → Solution → CTA**

- **Hook (0-3 sec):** Stop the scroll
- **Problem (3-8 sec):** Relatable pain point
- **Solution (8-15 sec):** Product demonstration
- **CTA (15-20 sec):** Clear next step

---

## Script 1: The Skeptic Turned Believer

**Best for:** Products with skeptical audiences, premium pricing

```
HOOK (0-3s):
"I'm not gonna lie, I thought this was just another overhyped [product category]..."

PROBLEM (3-8s):
"I've tried literally everything for [problem]. Spent hundreds on stuff that didn't work."

SOLUTION (8-15s):
"But after using [Product] for [timeframe], I'm actually shocked."
[Show product being used / results]
"Look at this [specific result]."

CTA (15-20s):
"Link's in bio. They have [offer] right now - honestly just try it."
```

**Video notes:** Handheld phone, close-up face, natural lighting, genuine tone

## Script 2: The Problem Call-Out

**Best for:** Products solving clear pain points

```
HOOK (0-3s):
"If you're still dealing with [specific problem], stop scrolling."

PROBLEM (3-8s):
"I know how frustrating it is when [describe frustration in detail].
You've probably tried [common solution] and it didn't work."

SOLUTION (8-15s):
"This is [Product]. Here's what makes it different..."
[Quick demo showing key feature]
"It actually [key benefit] because [simple reason]."

CTA (15-20s):
"They're doing [X]% off this week. Link in bio."
```

**Video notes:** Eye contact, confident delivery, show product in action

**Best for:** Lifestyle products, routines, consumables

```
HOOK (0-3s):
"POV: You finally found the [product] that actually works"

SCENE 1 (3-7s):
[Morning routine montage with product]
"Every morning I start with..."

SCENE 2 (7-12s):
[Using product naturally]
"What I love about it is [benefit]..."

SCENE 3 (12-18s):
[Results or satisfaction moment]
"And the best part? [Key differentiator]"

CTA (18-22s):
"Linked below if you want to try it"
```

**Video notes:** B-roll heavy, aesthetic shots, trending audio optional

## Script 4: The Before/After

**Best for:** Transformation products (skincare, fitness, cleaning, etc.)

```
HOOK (0-3s):
[Split screen or side-by-side]
"[Timeframe] with [Product]. Same lighting, no filter."

BODY (3-12s):
"Here's what I was dealing with before..."
[Show before state]
"And here's after [timeframe] of using [Product]..."
[Show after state]

PROOF (12-18s):
"I was honestly surprised because I've tried [competitors] and nothing worked like this."

CTA (18-22s):
"I'll leave the link below - they have a [guarantee/offer] so you can try it risk-free."
```

**Video notes:** Consistent lighting in before/after, genuine reactions

## Script 5: The Unboxing/First Impression

**Best for:** New product launches, premium products, subscription boxes

```
HOOK (0-3s):
"My [Product] finally came and I'm so excited"
[Show package arriving/opening]

UNBOXING (3-10s):
"Okay so first thing - the packaging is [observation]..."
[Show each item]
"And this is the [main product]..."

FIRST USE (10-18s):
"Let me try this right now..."
[Real-time reaction]
"Oh wow, okay so [immediate observation]..."

CTA (18-22s):
"I'll update you guys but first impression? [rating]. Link in bio."
```

**Video notes:** ASMR-friendly audio, close-up shots, genuine first reactions

## Script 6: The Comment Response

**Best for:** Addressing objections, building trust

```
HOOK (0-3s):
[Show comment overlay: "Those are so overpriced" or similar objection]
"Okay I get this comment a lot, let me explain..."

RESPONSE (3-12s):
"So yes, [Product] is [price], but here's what you're getting..."
[Show product value]
"You're paying for [quality/feature 1], [feature 2], and [feature 3]."

COMPARISON (12-18s):
"Compare that to [cheaper alternative] which is [problem with cheap option]."
"I actually did the math and [value justification]."

CTA (18-22s):
"If you want to see for yourself, link's in bio."
```

**Video notes:** Direct to camera, confident tone, have comment visible on screen

## Script 7: The Expert/Authority

**Best for:** Technical products, health/wellness, professional services

```
HOOK (0-3s):
"As a [profession/credential], here's what I think about [Product]..."

CREDIBILITY (3-8s):
"I've been [doing X] for [years] and I've tried pretty much everything."

ANALYSIS (8-15s):
"What makes [Product] different is [technical reason]."
[Show product/explain feature]
"Most [products in category] don't have this because [reason]."

VERDICT (15-20s):
"So from a [professional] perspective, this is [rating/recommendation]."
"Link below if you want to check it out."
```

**Video notes:** Professional but approachable, can use graphics/text for emphasis

---

## Script 8: The Comparison

**Best for:** Products with clear competitors, switching audiences

```
HOOK (0-3s):
"I switched from [Competitor] to [Product] and here's what happened..."

SETUP (3-8s):
"I used [Competitor] for [timeframe] and it was fine but [limitation]."

COMPARISON (8-15s):
"[Product] does [feature] which [Competitor] doesn't have."
[Side by side if possible]
"And the price is actually [comparison]."

VERDICT (15-20s):
"After [timeframe] I'm not going back. Link below."
```

**Video notes:** Fair comparison, specific not vague, show both products if possible


## Funnel-Specific Scripts

### Top of Funnel (TOF) - Awareness
Focus: Grabbing attention, relatability, entertainment

- Use: Scripts 2, 5, 9
- Tone: Casual, curious, discoverable
- CTA: Soft ("link in bio if curious")

### Middle of Funnel (MOF) - Consideration
Focus: Education, comparison, building trust

- Use: Scripts 1, 6, 7, 8
- Tone: Informative, credible
- CTA: Medium ("check it out")

### Bottom of Funnel (BOF) - Conversion
Focus: Social proof, urgency, overcoming objections

- Use: Scripts 3, 4, 10
- Tone: Confident, urgent
- CTA: Strong ("link below, limited time")

## Production Notes for Creators

**Technical:**
- Film vertical (9:16) for Reels/Stories
- Good lighting (natural preferred)
- Clear audio (reduce background noise)
- Keep to 15-30 seconds ideal

**Performance:**
- Sound natural, not scripted
- Make eye contact with camera
- Show genuine reactions
- Demonstrate don't just describe
