# Meta Ads Checklists

> Print these. Use them. Every campaign, every time.

---

## Campaign Launch Checklist

### Before You Launch

**Strategy**
- [ ] Clear campaign objective defined
- [ ] Target audience documented
- [ ] Budget set (10x target CPA minimum)
- [ ] Success metrics defined (CPA, ROAS target)
- [ ] Testing plan ready (what you're testing, why)

**Creative**
- [ ] 8-15 genuinely different ad concepts (not variations)
- [ ] Multiple formats tested (video + static + carousel)
- [ ] Hook optimized for first 3 seconds
- [ ] Mobile-first design (4:5 or 9:16 ratio)
- [ ] Captions/subtitles added (85%+ watch sound-off)
- [ ] No policy violations in copy or creative

**Copy**
- [ ] Headline under 40 characters
- [ ] Clear value proposition
- [ ] Single CTA per ad
- [ ] No banned words or claims
- [ ] Proofread for errors

**Technical**
- [ ] Pixel installed and verified
- [ ] Conversions API set up (server-side)
- [ ] Conversion event correct
- [ ] UTM parameters added
- [ ] Landing page tested on mobile

**Landing Page**
- [ ] Message matches ad promise
- [ ] Loads in under 3 seconds
- [ ] Mobile optimized
- [ ] Clear CTA above fold
- [ ] Social proof visible
- [ ] No distracting elements

## Weekly Optimization Checklist

**Performance Review (Monday)**
- [ ] Review key metrics vs. last week
- [ ] Check CTR trends (target: 0.9-1.6%)
- [ ] Review CPA/ROAS vs. target
- [ ] Check frequency levels (<3-4 ideal)
- [ ] Document any anomalies

**Creative Health (Tuesday)**
- [ ] Identify fatigued ads (declining CTR, rising CPC)
- [ ] Pause underperformers (no results after $50-100 spend)
- [ ] Scale winners (increase budget 20% max per change)
- [ ] Review new creative needs
- [ ] Check competitor activity in Ad Library

**Budget & Bidding (Wednesday)**
- [ ] Review spend pacing
- [ ] Reallocate budget to top performers
- [ ] Check for overspending ad sets
- [ ] Review bid strategy performance
- [ ] Adjust if CPA trending wrong direction

**Audience Check (Thursday)**
- [ ] Review breakdown by age, gender, placement
- [ ] Check for audience overlap issues
- [ ] Evaluate audience saturation
- [ ] Consider expansion opportunities
- [ ] Update exclusions if needed

**Technical Audit (Friday)**
- [ ] Verify tracking is firing correctly
- [ ] Check for any policy warnings
- [ ] Confirm payment method current
- [ ] Review Meta notifications
- [ ] Document learnings for the week

## Creative Testing Checklist

**Before Testing**
- [ ] Clear hypothesis defined ("I believe X will outperform Y because Z")
- [ ] Single variable isolated (change ONE thing)
- [ ] Sufficient budget allocated ($50-100+ per variant)
- [ ] Success metric defined
- [ ] Kill criteria established (when to stop)
- [ ] Timeline set (minimum 5-7 days)

**What to Test**
- [ ] Hook (first 3 seconds or headline)
- [ ] Format (video vs static vs carousel)
- [ ] Angle (problem vs benefit vs social proof)
- [ ] CTA (Shop Now vs Learn More vs custom)
- [ ] Creator (different UGC creators)
- [ ] Length (7 sec vs 15 sec vs 30 sec)

**After Testing**
- [ ] Statistical significance reached?
- [ ] Results documented
- [ ] Learnings extracted
- [ ] Winner moved to scale
- [ ] Losers killed
- [ ] Next test planned

## Scaling Checklist

**Before Scaling**
- [ ] Consistent performance for 5+ days
- [ ] CPA at or below target
- [ ] Learning phase exited
- [ ] Frequency still healthy (<3)
- [ ] Audience has room to expand

**Scaling Actions**
- [ ] Increase budget 20% maximum per change
- [ ] Wait 2-3 days between increases
- [ ] Monitor CPA closely after each change
- [ ] Have backup creative ready
- [ ] Document scaling velocity

**If Performance Drops**
- [ ] Reduce budget to previous level
- [ ] Check creative fatigue
- [ ] Review audience saturation
- [ ] Consider horizontal scaling (new audiences)
- [ ] Test fresh creative

## BFCM Preparation Checklist

### 90 Days Out (September)
- [ ] Finalize promotional strategy
- [ ] Set revenue/ROAS targets
- [ ] Brief creative team
- [ ] Book UGC creators
- [ ] Plan audience warm-up strategy

### 60 Days Out (October)
- [ ] Complete creative production
- [ ] Begin audience warming
- [ ] Test messaging/offers
- [ ] Set up campaign structure (drafts)
- [ ] Confirm inventory levels

### 30 Days Out (November)
- [ ] Final creative review
- [ ] Campaigns ready to launch
- [ ] Team roles assigned
- [ ] Monitoring schedule set
- [ ] Backup creative prepared

### Week Of
- [ ] Launch campaigns
- [ ] Monitor hourly during peak days
- [ ] Refresh creative daily
- [ ] Adjust budgets in real-time
- [ ] Document everything for next year

## Ad Troubleshooting Quick Reference

| Problem | First Check | Quick Fix |
|---------|-------------|-----------|
| Not delivering | Budget/bid | Increase budget, remove caps |
| Low CTR | Hook/creative | Test new hooks |
| High CPC | Audience | Broaden targeting |
| Low CVR | Landing page | Speed + message match |
| High CPA | All of above | Start with creative |
| Declining performance | Fatigue | Fresh creative |
| Learning Limited | Data volume | Higher funnel event |

## End of Month Review Checklist

**Performance Analysis**
- [ ] Review monthly spend vs. budget
- [ ] Calculate blended ROAS/CPA
- [ ] Compare to previous month
- [ ] Identify top performing campaigns
- [ ] Identify underperformers

**Creative Insights**
- [ ] What creative worked best?
- [ ] What hooks drove highest CTR?
- [ ] What angles resonated?
- [ ] What format performed best?

**Audience Insights**
- [ ] What audiences performed best?
- [ ] Any demographic trends?
- [ ] Placement performance differences?
- [ ] Device performance differences?

**Next Month Planning**
- [ ] Set next month targets
- [ ] Plan creative needs
- [ ] Document hypotheses to test
- [ ] Update strategy based on learnings
